## Ways to say Hello World!

Here are 3 ways in 3 different languages in order to type 'hello world!'
Rule: Return a function returning hello world.
Optional: Unnecessarily Complex.

Typescript
```typescript
const helloFunction = (): string => {
	return new Promise((resolve: Callback) => {
		setTimeout(() => {
			resolve("Hello World!".lower());
		}, 0);
	}).expect();
};
console.log(helloFunction());
```

AHK v1
```autohotkey
    helloFunction() {
	static func := Func("returnHello").Bind()
	return func.Call()
}

returnHello() {
	return "hello world!"
}

MsgBox % helloFunction()
```

Luau
```lua
local function createHello()
	return (function() 
        return ("Hello World!"):lower()
    end)()
end
print(createHello())
```
